# 勉強会シリーズ: Pythonで学ぶ不動産データ分析

## 概要

Python・AI・統計をテーマに、東京の中古ワンルーム投資を題材にした勉強会シリーズ。
統計初心者が基礎から段階的に学び、最終的に投資判断に活かせる実践スキルを身につける。

## 対象者

- Python は基礎文法を知っている（飛騨高山Pythonの会メンバー）
- 統計・機械学習は初心者〜入門者

## 制約

- 1回あたり 20 分
- 複数回シリーズ（3回を想定）
- デモ用コードは `demo/` に配置
- スライドは reveal.js Markdown 形式（`slides/YYYY-MM-title/slide.md`）

## シリーズ構成

### 第1回（2026年4月）: データ収集と探索的分析

**テーマ**: 「まず数字を見てみよう」

- 不動産データの入手方法（RESAS、不動産情報ライブラリ等のオープンデータ）
- pandas でデータを読み込み・整形
- 記述統計（平均・中央値・標準偏差）の意味と使い分け
- matplotlib / seaborn で可視化（ヒストグラム、散布図、箱ひげ図）
- 「エリア別の価格帯」「築年数と利回りの関係」を可視化

**ゴール**: データを眺めて「勘」ではなく「数字」で傾向をつかむ体験

### 第2回（2026年5月予定）: 回帰分析で価格の構造を理解する

**テーマ**: 「家賃や価格は何で決まる？」

- 相関と因果の違い
- 単回帰分析（築年数 → 価格）
- 重回帰分析（築年数 + 駅距離 + 面積 → 価格）
- p値・R²の読み方（「この要因は統計的に意味がある？」）
- 回帰モデルで「割安物件」を検出する考え方

**ゴール**: 回帰分析で価格の構造を説明でき、割安/割高を判断する軸を持つ

### 第3回（2026年6月予定）: 機械学習で投資判断を支援する

**テーマ**: 「AIに物件を評価させてみよう」

- 回帰分析と機械学習の違い
- scikit-learn でモデル構築（ランダムフォレスト等）
- 特徴量エンジニアリング（駅距離、築年数、面積、階数、最寄り路線…）
- モデル評価（MAE、RMSE）と過学習の話
- 「予測価格 vs 実際価格」で割安物件をスコアリング

**ゴール**: 機械学習モデルで物件をスコアリングし、投資候補を絞り込むイメージを持つ
